## Учеьный проект "Месседжер". Яндекс.Практикум.

Реализация месседжера. Превый спринт курса "мидл-фронтенд".

## Установка

- `npm install` — установка.
- `npm run build` — сборка.
- `npm start` — сборка + запуск (express).
